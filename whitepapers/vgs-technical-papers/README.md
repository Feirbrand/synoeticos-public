# VGS Technical Papers

**Public technical papers for the ValorGrid / Synoetic OS research ecosystem.**

This directory contains DOI-backed technical papers and selected supporting research documents for AI resilience, memory architecture, recovery systems, symbolic stability, context engineering, and threat-aware cognitive systems.

For canonical DOI metadata, use [`../../PUBLICATIONS.md`](../../PUBLICATIONS.md).

---

## Public Scope

This folder may include:

- Public paper sources
- Public methodology notes
- Selected supporting technical documents
- Citation and DOI links

This folder does not include:

- Runnable framework code
- Raw configs or workflow files
- Private benchmark harnesses
- Internal adapters
- Production deployment playbooks
- Private threat-intelligence indicators
- Pricing, service, support, training, or certification material
