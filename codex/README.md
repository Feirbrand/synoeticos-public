# Threat Resilience Codex

**Public research map for AI behavioral failure patterns, symbolic instability, and recovery-aware resilience.**

[![License](https://img.shields.io/badge/License-CC--BY--NC--4.0-green)](https://creativecommons.org/licenses/by-nc/4.0/) [![DOI](https://img.shields.io/badge/DOI-10.5281%2Fzenodo.17451060-blue)](https://doi.org/10.5281/zenodo.17451060)

The Threat Resilience Codex is an open research effort documenting observable patterns of drift, desynchronization, and identity-level failures in large language models and cognitive systems. This repository contains selected public materials only. The full operational threat library and defense playbooks are maintained internally in Notion/Obsidian.

**Published Paper:** DNA Codex v5.5 · DOI: [10.5281/zenodo.17451060](https://doi.org/10.5281/zenodo.17451060)
