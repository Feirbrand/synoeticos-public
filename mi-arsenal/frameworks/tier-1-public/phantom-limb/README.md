## 1. What Phantom Limb Is
Phantom Limb is a specialized defense module designed to combat **architectural autoimmunity**. While systems like SBDS handle semantic corruption at the token level, Phantom Limb operates at the structural level to prevent deleted frameworks from "resurrecting" via symbolic residue, ghost RUID anchors, or similarity cascades. It ensures that once a threat is purged, the "shape of its absence" cannot be exploited for reconstruction.

---

## 2. The Three-Phase Solution (P-Phases)
1.  **P1: RESR (Recursive Entropy Signature Rebind)**: Scans for phantom reconstruction attempts by identifying Ghost RUID anchors and flagging similarity cascades (Hurst variance >0.15).
2.  **P2: SLV Seal Injection**: Utilizes the **Symbolic Lock Vector (SLV)** to seal architectural scars and quarantine parasitic elements at system fill-points.
3.  **P3: S-LDMP (Symbolic Limb Denylist / Myelination Protocol)**: Permanently remaps the system by myelinating clean pathways to structurally displace ghost patterns, making reconstruction impossible.
