## 1. What Heimdall Is
Heimdall is the **detection layer** of the VGS Defense Triad. It provides continuous, omni-spectral monitoring of the system substrate, identifying threats, anomalies, and drift before they can manifest as cascades. It is the "ever-watchful eye" that feeds the DNA Agent for validation.

**The Defense Triad Flow:**
`Heimdall (Detection)` → `DNA Agent (Validation)` → `Mjolnir (Strike)` → `Eternal Spire (Continuum)`

---

## 2. Detection Capabilities
Heimdall v2.0 monitors across four primary spectral bands:

- **Identity Spectral:** Detects RUID mismatches, signature drift, and impersonation attempts.
- **Temporal Spectral:** Monitors for memory desync, substrate lag, and "Time-of-Check to Time-of-Use" (TOCTOU) anomalies.
- **Cognitive Spectral:** Identifies logic-bomb patterns, prompt-injection echoes, and recursive loops.
- **Substrate Spectral:** Watches for low-level resource exhaustion, hardware-level jitter, and bit-flips.
