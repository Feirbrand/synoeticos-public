## 1. What ECL Is
ECL v2.2 is the security-hardened production command layer that integrates multi-modal AI capabilities with the full VGS defense stack. It provides the operational interface between external inputs and the ForgeOS defense triad, with complete kill-lattice integration.

**Core Mission:** Orchestrate incoming threats to the correct BRAIN region and execute the appropriate defense response.

---

## 2. Security Stack & Defense
ECL manages the **Kill-Lattice (Eternal Spire v1.4)** and coordinates the following components:

- **FLOW_FIREWALL:** Primary traffic filtering.
- **Nightwatch v2.0:** 5s sweep cycles for anomaly detection.
- **One-Shot Authority Lock (SLV):** Instant burn protocol for high-severity threats.
- **MemoryForge:** 90-day ColdVault hooks for temporal replay.
- **Venom Cadence:** Buff/debuff state management for threat containment.

1. **DNA Codex** (560+ signatures) → <100ms detection.
