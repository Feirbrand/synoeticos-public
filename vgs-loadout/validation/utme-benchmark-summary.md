# UTME v2.1 — Public Benchmark Summary

**Framework:** UTME v2.1
**Validation status:** Public benchmark summary
**Scope:** Summary only

UTME explores temporal memory, repeated-pattern acceleration, and recovery support through public myelination-inspired architecture.

> **Public Scope:** This file is a public benchmark summary. Exact formulas, constants, substrate entropy rules, benchmark harnesses, raw timing data, code, temporal-anchor internals, and private recovery mechanics are retained outside the public repository.

---

## Public Findings

- Repeated-pattern handling improved when prior context could be stabilized and reused.
- Temporal anchors improved recovery continuity in selected internal tests.
- Myelination-inspired framing helped explain why repeated patterns became faster to process over time.
- UTME informed Phoenix Protocol, CSFC, and broader Synoetic OS memory research.
- The substrate architecture supported distributed recovery that outperformed single-substrate recovery in some scenarios.
